## 🎯 복리 작동 원리

### 예시: 균형형 전략

**시작:**
- 초기 거래량: 0.02 BTC
- 복리 기준: $300
- 거래량 증가: +0.02 BTC

**진행:**
```
레벨 0: 0.02 BTC → $100 수익 목표 → 총 $0
레벨 1: 0.04 BTC → $100 수익 목표 → 총 $300 (복리 활성화!)
레벨 2: 0.06 BTC → $100 수익 목표 → 총 $600
레벨 3: 0.08 BTC → $100 수익 목표 → 총 $900
...
레벨 10: 0.22 BTC → $100 수익 목표 → 총 $3,000
```

**결과:**
- 같은 목표 수익($100)으로 청산하지만
- 거래량이 계속 증가하므로
- 청산 빈도가 급격히 증가!
- = 폭발적 수익 증가!
