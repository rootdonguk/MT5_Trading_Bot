🎮 완벽한 수동 제어 기능 추가!
⌨️ 키보드 명령어
H 키 - 수익만 청산 (파란불 💙)
MT5에서 파란불 (수익 포지션)만 모두 청산하고 종료
손실 포지션은 그대로 유지
L 키 - 손실만 청산 (빨간불 ❤️)
MT5에서 빨간불 (손실 포지션)만 모두 청산하고 종료
수익 포지션은 그대로 유지
Q 키 - 전체 청산
모든 포지션 청산하고 종료
S 키 - 통계 보기
현재 상태 확인
- 수익 포지션 개수 및 금액
- 손실 포지션 개수 및 금액
- 누적 수익, 방향전환 횟수 등
C 키 - 계속 실행
봇 계속 실행

📊 실시간 화면 예시
실행 중:
[14:23:45] BTC: $50,123.45 | 💙12 ❤️3 | 수익: +$2.45

⌨️ 키보드 명령
H = 수익 포지션만 청산 (파란불 💙)
L = 손실 포지션만 청산 (빨간불 ❤️)
Q = 모든 포지션 청산
S = 통계 보기

프로그램 종료

🎯 사용 시나리오
시나리오 1: 수익 실현하고 퇴근
상황: 오후 6시, 퇴근 시간
포지션: 💙 20개 수익, ❤️ 5개 손실

행동:
1. S 키 눌러서 현재 상태 확인
2. 수익 포지션: $0.002 (좋음!)
3. H 키 입력
4. 수익 포지션만 청산하고 종료
5. 손실 포지션은 내일 회복 기대
시나리오 2: 손실 정리
상황: BTC 급락 중
포지션: 💙 3개 수익, ❤️ 30개 손실

행동:
1. 방향전환이 작동 중이지만
2. 더 이상 기다리고 싶지 않음
3. L 키 입력
4. 손실 포지션만 정리
5. 수익 포지션은 계속 유지
시나리오 3: 긴급 종료
상황: 중요한 뉴스 발표 직전
포지션: 💙 15개, ❤️ 15개

행동:
1. Q 키 입력
2. 모든 포지션 즉시 청산
3. 안전하게 종료
시나리오 4: 상태만 확인
상황: 궁금해서 확인
포지션: 💙 10개, ❤️ 2개

행동:
1. S 키 입력
2. 통계 확인
3. C 키 입력
4. 계속 실행

💡 핵심 장점
1. 선택적 청산

수익만 실현하고 손실은 회복 대기
손실만 정리하고 수익은 더 키우기

2. 실시간 확인

💙 파란불 개수 실시간 표시
❤️ 빨간불 개수 실시간 표시

3. 빠른 대응

키 하나로 즉시 청산
복잡한 조작 불필요

4. 안전한 종료

모든 대기 주문 자동 취소
